# FlexFinToolkit — Financial Analysis Suite

Upload AR, AP, and GL data to get instant financial analysis with 12-month forecasts, scenario modeling, and executive reports.

## Getting Started

```bash
npm install
npm run dev
```

## Features

- CSV Upload with auto-column detection
- 12-month cash flow forecast
- Scenario analysis (Base/Best/Worst)
- Working capital analysis (DSO/DPO/CCC)
- Budget vs actual variance
- Excel export
- Executive summary report generation
